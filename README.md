## ✔️ 개발 동기
강의 영상은 열람 기간이 지나면 삭제되고, 완강하기 전에는 사용자가 강의 영상의 타임라인을 조절할 수 없는 등의 불편함이 있습니다. 따라서 강의 영상을 다운로드하여 시청하는 학생이 많습니다.

동서대학교 기준, 2021년 1학기까진 강의 영상 링크를 브라우저에 입력해 다운로드할 수 있었습니다. 그러나 2021년 2학기부터 Learning-X의 시스템이 변경되면서 영상을 다운로드할 수 없게 되었습니다.

저는 이 문제를 해결하기 위해 New LMS Downloader 프로그램을 제작하게 되었습니다.
